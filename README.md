# Apple Style E-Commerce Catalog Page Project
This is a showcase of a traditional hand-coded frontend website for LearnFrontEndNow. 

The Build Process
This project was designed to mirror the clean, minimalist aesthetic of Apple’s product pages. Unlike using a drag-and-drop builder, this site was hand-coded from scratch to ensure pixel-perfect precision and high performance.

## Tech Stack
- HTML5: Structured for accessibility and SEO.

- SCSS: Used for modular styling, leveraging variables and nesting to keep the design system consistent and maintainable.

- JavaScript: Handles the interactive elements and UI logic.

- Vite: Utilized as the frontend build tool to provide a lightning-fast development environment and optimized production bundles.


Building with Vite allowed me to leverage a modern development workflow while maintaining the integrity of a "vanilla" project. By using SCSS instead of standard CSS, I was able to manage complex layouts (like the product grid and responsive navigation) with much cleaner, more organized code.
